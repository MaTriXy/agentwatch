# Contributing

Thank you for considering contributing to spyllm! We welcome contributions to improve this project. For general contributions and community guidelines, please see the [repo](https://github.com/cyberark/community/).
## General Steps for Contributing

1. Fork the project.  
2. Clone your fork.  
3. Make local changes to your fork by editing files.  
4. Commit your changes.  
5. Push your local changes to the remote server.  
6. Create a new Pull Request.  

From here, your pull request will be reviewed, and once you've responded to all feedback, it will be merged into the project. Congratulations, you're a contributor!
